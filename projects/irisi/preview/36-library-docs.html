Editorial broadsheet. Slides that read like a serious newspaper.

A measured, byline-and-rule pack for board updates, research summaries, intelligence briefs, and any deck that wants to feel edited rather than generated. Twelve layouts, fully typeset.

— Stance

"What if a NYT analyst designed your deck?"

Bylines, horizontal rules, kicker-coded sections, two-column body, restrained pull-quotes. One journalistic red used only on critical or breaking-news cues.

Layouts: 12 Fonts: Source Serif 4 · Inter Best for: board, research, analysis
— Quarterly · Q2 2026 —
The quiet rebellion of measured growth.
By the founders · 12 min read

01 Installation

Add Irisi's slide library to your project. The library is provider-agnostic — bring your own React app, Next.js page, or render to HTML/PDF on the server.

terminal 
# pick your package manager
npm install @irisi/slides @irisi/pack-editorial
# or
pnpm add @irisi/slides @irisi/pack-editorial

02 Quickstart

Render an editorial-broadsheet headline slide with content. Each layout is a typed React component; props mirror what the AI fills in.

tsx app/deck/page.tsx 
import { Headline } from "@irisi/pack-editorial";

export default function Page() {
  return (
    <Headline
      kicker="— Quarterly · Q2 2026 —"
      title="The quiet rebellion of measured growth"
      byline="By the founders · 12 min read"
      paragraphs={[
        "Most companies confuse motion for progress…",
        "The reason isn't strategy. It's discipline.",
      ]}
      brand={{ accent: "#B42318", mark: "H" }}
      page={{ index: 3, total: 12 }}
    />
  );
}

04 The Headline component, in detail

Props for <Headline> — the most-used layout in this pack. All props are optional except title; sensible defaults keep the typesetting consistent.

PropTypeDescription
titlestringThe display headline. Long titles wrap inside the editorial column; the layout will not truncate.
kickerstringThe small uppercase byline above the title — typically issue, quarter, or section.
bylinestringThe reading-time / authorship line beneath the rule.
paragraphsstring[]Body copy. Two paragraphs maximum is recommended for legibility at projection size.
brandBrandTokensColour, mark, and typography overrides. Inherited from <DeckProvider> by default.
page{index, total}For the footer's page-numbering. Auto-supplied by <Deck> when used as a child.

05 For AI agents

Each layout publishes a JSON schema and a tool-call definition. Hand them to your function-calling LLM and it'll produce valid props directly.

json headline.schema.json 
{
  "name": "editorial.Headline",
  "description": "A typeset editorial headline slide with kicker, title, byline, and two-column body.",
  "input_schema": {
    "type": "object",
    "required": ["title"],
    "properties": {
      "title": { "type": "string", "maxLength": 90 },
      "kicker": { "type": "string", "maxLength": 42 },
      "paragraphs": {
        "type": "array", "items": { "type": "string" }, "maxItems": 2
      }
    }
  }
}

Avatars · chips · verified

The trust idiom

Avatars are warm-toned gradients with a hairline edge. They never carry a status dot or a checkmark badge — the verified capsule does that work, separately, where it can be read. The masked username convention shows only the first name and last initial; full names are stored privately for the verification team.

Scene · avatars · tones

Four warm gradients · used to vary a stack

When eight tutors stack in a search result we want the page to feel populated, not uniform. Four neutral, warm tones do the rotation. Never team colours, never personality colours.

Scene · the masked-username idiom

Public name · first + initial only

Tutorly stores the full legal name privately. Publicly, only the first name and last initial appear — a small, deliberate signal that this is a person, not a transaction. We never show a username, handle, or surname in chrome.

Scene · the verified capsule · the only trust mark

One trust mark. Repeated everywhere.

Used on profiles, in conversation headers, on credentials, on documents. Always the same shape. Always the same words. A parent who sees it three times reads the system as consistent.
